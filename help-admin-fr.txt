** -- Info Commands --
**   HELP                 : Voir l'Aide
**   XDLFULL              : Liste tous les packs proposés
**   XDLGROUP [group]     : Voir les packs de <groupe>
**   XDL                  : Liste les fichiers par groupes et les fichiers sans groupe
**   XDLOCK               : Voir tous les packs bloqués
**   XDTRIGGER            : Voir tous les packs avec des déclencheurs dynamiques
**   FIND pattern         : Liste des packs qui correspond à <modèle>
**   XDS                  : Sauvegarde du fichier statistiques
**   DCL                  : Liste les transferts en cours
**   DCLD                 : Liste les transferts en cours avec détails
**   TRINFO id            : Liste les informations du transfert <id>
**   GETL                 : Liste des upload en attente
**   QUL                  : Liste la file d'attente
**   IGNL                 : Voir la liste des ignore
**   LISTUL [dir]         : Voir le contenu du répertoire d'upload
**   DISKFREE             : Voir l'espace libre du répertoire d'upload
**   CHANL [net]          : Voir la liste des chans et users
** -- Transfer Commands --
**   CLOSE [id]           : Coupe le transfert <id>
**   CLOSEU [id]          : Coupe l'upload <id>
**   ACCEPTU min [hostmask] : Accepte les uploads de <hostmask> pour <x> minutes
**   GET net nick n [password] : Demander le pack <n> du bot <pseudo>
**   CLOSEGET net nick    : Annule la demande de pack pour le bot <pseudo>
**   RMQ [position]       : Supprime l'entrée <position> de la file d'attente
**   RMIQ [position]      : Supprime l'entrée <position> de la file d'attente d'inactivité
**   RMALLQ               : Enlèves les entrées de la file d'attente d'inactivité et de la file d'attente normale
**   NOMIN id             : Désactive la vitesse minimum du transfert <id>
**   NOMAX id             : Désactive la vitesse maximum du transfert <id>
**   UNLIMITED id         : Désactive les limites de bande passante pour le transfert <id>
**   MAXSPEED id x        : Définit la limite max de bande passante <x> KB/s du transfert <id>
**   SEND nick n [net]    : Envoi le pack <n> à <nick>
**   QUEUE nick n [net]   : Met le pack <n> pour <nick> en file d'attente
**   IQUEUE nick n [net]  : Met le pack <n> pour <pseudo> en file d'attente d'inactivité
**   BATCH nick n-m [net] : Met les packs <n> à <m> pour <pseudo> en file d'attente d'inactivité
**   PSEND channel style [net] : Envoi <style (full|minimal|summary)> (complet|minimal|résumé) XDCC LIST sur <chan>
**   QSEND [id]           : Démarrer le transfert d'une file d'attente
**   IQSEND [id]          : Passer un envoi de la file d'attente d'inactivité à la file d'attente normale
**   SLOTSMAX [slots]     : Changement temporaire de slotsmax à x <slots>
**   QUEUESIZE [slots]    : Changement temporaire de taille de la file d'attente à x <slots>
**   REQUEUE x y          : Change la position de la file d'attente <x> à <y>
**   REIQUEUE x y         : Change l'entrée de la file d'attente d'inactivité de la position <x> à <y>
** -- Pack Commands --
**   INFO n               : Voir les informations du pack <n>
**   REMOVE n [m]         : Enlève le pack <n> ou les packs de <n> à <m>
**   REMOVEDIR dir        : Enlève tous les packs trouvés dans <dir>
**   REMOVEGROUP group    : Enlève tous les packs trouvés dans <groupe>
**   REMOVEMATCH pattern  : Enlève tous les packs contenant ce terme
**   RENUMBER x [y] z     : Déplace les packs de <x> à <y> à la position <z>
**   SORT [field] [field] : Trie les packs par <champ>. Option : NAME, DESC, GROUP, PATH, SIZE, TIME, ADDED (Nom, Description, Groupe, Chemin, Taille, Temps, Ajout)
**   ADD fichier          : Ajoute le fichier <fichier>
**   ADDDIR dir           : Ajoute les fichiers de <dir>
**   ADDNEW dir           : Ajoute les nouveaux fichiers de <dir>
**   ADDGROUP groupe dir  : Ajoute les nouveaux fichiers de <dir> dans <groupe>
**   ADDMATCH pattern     : Ajoute les nouveaux fichiers correspondant à <modèle>
**   AUTOADD              : scan autoadd_dirs pour des nouveaux fichiers
**   AUTOCANCEL           : Annule et retire les actions des ajouts
**   AUTOGROUP            : Créé un groupe pour chaque répertoire avec des packs
**   NOAUTOADD x          : Désactive AUTOADD pour les <x> prochaines minutes
**   CHFILE n nouveau_nom : Change le nom du pack <n> en <nouveau_nom>
**   NEWDIR dirname newdir : Renomme le répertoire de tous les packs correspondant
**   CHDESC n [msg]       : Change la description du pack <n> en <msg>
**   CHNOTE n [msg]       : Change la note du pack <n> en <msg>
**   CHTIME n [msg]       : Change la date d'ajout du pack <n> en <msg>
**   CHMINS n x           : Change la vitesse mini du pack <n> à <m> en <x> KB/s
**   CHMAXS n x           : Change la vitesse maxi du pack <n> à <m> en <x> KB/s
**   CHLIMIT n x          : Change la vitesse limite de download du pack <n> à <m> en <x> transferts par jour
**   CHLIMITINFO n [msg]  : Change l'annonce de la vitesse limite de download du pack <n> en <msg>
**   CHTRIGGER n [msg]    : Change le trigger du pack <n> en <msg>
**   DELTRIGGER n [m]     : Efface le trigger du pack <n> à <m>
**   CHGETS n x           : Change le nombre de DL du pack <n> à <m> en <x>
**   CHCOLOR n [m] x[,b][,s] : Définit le pack <n> à <m> en couleur <x> avec un fond <b> et un style <s>
**   LOCK n [m] password  : Bloque les packs <n> à <m> avec <password>
**   UNLOCK n [m]         : Débloque les packs <n> à <m>
**   LOCKGROUP group password : Bloque tous les packs de <groupe> avec <password>
**   UNLOCKGROUP group    : Débloque tous les packs de <groupe>
**   RELOCK ancien_password password : Bloque tous les packs avec <ancien_password> par <password>
**   GROUPDESC group msg  : Change la description de <groupe> en <msg>
**   GROUP n group        : Change le groupe du pack <n> en <groupe>
**   MOVEGROUP n m group  : Déplace le groupe vers <groupe> pour les packs <n> à <m>
**   REGROUP group nouveau : Change tous les packs de <groupe> en groupe <nouveau>
**   NEWGROUP groupe dir  : Change tous les packs de <dir> vers <groupe>
**   ANNOUNCE n [msg]     : Annonce <msg> pour pack <n> dans tous les chans joint
**   NEWANN n [channel] [net] : Annonce le(s) dernier(s) <n> packs dans tous les chans joint
**   MANNONCE n m [msg]   : Annonce <msg> pour <n> à <m> dans tous les chans joint
**   CANNOUNCE channel n [msg] : Annonce <msg> pour le pack <n> dans <chan>
**   SANNOUNCE n [m]      : Annonce courte pour les packs <n> à <m> dans tous les chans joint
**   NOANNOUNCE x         : Désactive toutes les annonces pour les <x> prochaines minutes
**   ADDANN fichier       : Rajoute et annonce un nouveau pack
**   MD5 [n] [m]          : Calcule MD5 et CRC32 des packs <n> à <m>
**   CRC [n] [m]          : Vérifie le CRC des packs <n> à <m>
** -- File Commands --
**   FILEMOVE fichier nouveau_nom : Renomme le fichier sur le disque
**   MOVEFILE n nouveau_nom : Renomme sur le disque le fichier <n> en <nouveau_nom>
**   MOVEGROUPDIR groupe dir : Déplace tous les fichiers de <groupe> vers <dossier>
**   FILEDEL fichier      : Efface le fichier sur le disque
**   FILEREMOVE n [m]     : Enlève le pack <n> ou de <n> à <m> et l'efface du disque
**   SHOWDIR dir          : Liste le contenu de <dir> sur le disque
**   MAKEDIR dir          : Créer un nouveau répertoire sur le disque
**   FETCH file url       : download l'url et sauvegarde le fichier dans le répertoire d'upload
**   FETCHCANCEL [id]     : stop le download du fichier de l'url <id>
** -- Misc Commands --
**   MSG nick message     : Envoi <msg> à l'user ou sur le chan <nick>
**   AMSG msg             : Annonce <msg> sur tous les chan joints
**   MSGNET net nick message : Envoi <msg> à l'user ou sur le chan <nick>
**   MESG message         : Envoi <msg> à tous les users en cours de transfert
**   MESQ message         : Envoi <msg> à tous les users en file d'attente
**   IGNORE x hostmask    : Ignore <hostmask> (nick!user@host) pour <x> minutes, * autorisé
**   UNIGNORE hostmask    : N’Ignore plus <hostmask>
**   BANNHOST x hostmask  : Stop transferts et ignore <hostmask> (pseudo!user@host) pour <x> minutes
**   BANNNICK nick [net]  : Stop transferts et enlève <pseudo> de la file d'attente
**   NOSAVE x             : Désactive l'autosave pour les prochaines <x> minutes
**   NOSEND x [msg]       : Désactive XDCC SEND pour les prochaines <x> minutes
**   NOLIST x             : Désactive XDCC LIST et PLIST pour les <x> prochaines minutes
**   NOMD5 x              : Désactive le calcul MD5 et CRC pour les <x> prochaines minutes
**   MSGREAD              : Voir MSG log
**   MSGDEL               : Efface MSG log
**   RMUL fichier         : Efface un fichier dans le répertoire d'upload
**   RAW commande         : Envoi <commande> aux serveurs (RAW IRC)
**   RAWNET net commande  : Envoi <commande> aux serveurs (RAW IRC)
**   LAG [net]            : Voir le lag sur les réseaux
** -- Bot Commands --
**   SERVERS [net]        : Voir la liste des serveurs
**   HOP [channel] [net]  : Part et rejoint un chan pour obtenir le statut
**   NOCHANNEL x [channel] : Quitter le chan pour <x> minutes
**   JOIN channel [net] [key] : Rejoignez le chan jusqu'au prochain rehash
**   PART chan [net]      : Quitte le chan jusqu'au prochain rehash
**   JUMP Serveur [net]   : Basculer vers un autre serveur ou vers <serveur>
**   SERVQC               : Efface la file d'attente du serveur
**   STATUS               : Voir les Informations utiles
**   REHASH               : Relire le fichier de configuration et reconfigure
**   BOTINFO              : Voir les informations sur le statut du Bot
**   MEMSTAT              : Voir les informations sur l'usage de la mémoire
**   CLEARRECORDS         : Efface transferts, bande passante, temps de fonctionnement, total envoyé, et limite de transfert
**   CLEARGETS            : Efface les compteurs de téléchargement pour chaque pack
**   REDRAW               : Redessine l'écran
**   DELHIST              : Efface l'historique de la console
**   QUIT                 : Ferme ce DCC chat
**   EXIT                 : Ferme ce DCC chat
**   LOGOUT               : Ferme ce DCC chat
**   CHATME               : Vous envoi une requête DCC Chat
**   CHATL                : Liste les informations DCC chat
**   CLOSEC [id]          : Ferme le DCC chat <id>
**   DEBUG x              : Fixe le niveau de debug à <x>
**   CONFIG Valeur de la clé : Définir la variable de configuration <key> à <value>
**   PRINT key            : Affiche la variable de configuration <key>
**   IDENTIFY [net]       : Renvoyer le mot de passe stocké à nickserv
**   HOLDQUEUE [x]        : Réglage de la file d'attente, x=1 set, x=0 reset
**   OFFLINE [net]        : Fermer le réseau indiqué ou tous les réseaux
**   ONLINE [net]         : Se connecter au réseau indiqué ou à tous les réseaux
**   RUBY method [args]   : Appel une méthode dans le ruby_script avec les paramètres <args>
**   SHUTDOWN act         : Fermeture d'iroffer, <act> est "now" maintenant, "delayed" retardé, ou "cancel" annulé
**   BACKGROUND           : basculer en arrière plan
**   DUMP                 : Ecrire un dump dans le fichier log
**   RESTART              : Arrêter et redémarrer le bot
**   CRASH                : Ecrire un dump dans le fichier log et sortir
